Localizing Tanjunka

The simple explanation is if you have Microsoft Excel, load up the file langs\tanjunka-language-en-us.xml then pick "SaveAs" and save it in the same folder with the name changed for your language. For example if you are doing a German translation you might name it "tanjunka-language-deutsch.xml"

Now edit the middle column. Most importantly Cell B2, enter a new LanguageLabel, for example "Deutsch".

The next time you run Tanjunka your new language should be available in the preferences. (Tools->Preferences).  Any thing you do not translate will come from the English file (tanjunka-language-en-us.xml).

If you do not have Microsoft Excel you can instead use Open Office which you can download for free from http://openoffice.org. Unfortunately Open Office 1.1 does not support Excel's XML format so there will be no way for you to get your language into Tanjunka directly but each .xml file is accompanied by an original .xls file  You can edit that file in Open Office, save it, send it to me and I can convert it for the next release of Tanjunka.

If you make a new language file for Tanjunka please contact me in the Forums at Tanjunka.com and I'll try to include it in the next release of Tanjunka.
